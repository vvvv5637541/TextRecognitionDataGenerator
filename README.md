# TextRecognitionDataGenerator
## Introduction
生成繁體中文文字圖像以供訓練。
## Build Process
* 首先先下載pakage，就可以開始操作。

```pip install trdg```

並執行

```pip install -r requirements.txt```

安裝其他需要的pakage。
## Basic CLI
首先先cd進入trdg資料夾。
### 生成繁體中文圖像
如要隨機生成五個中文字的圖片，可簡單執行
```python run.py -c 300 -l cn -w 5```
